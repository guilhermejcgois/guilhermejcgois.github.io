ShoppingList

Este app permite gerenciar suas listas de compra online para você não esquecer produtos durante as compras ou tenha que perder tempo procurando procurando suas listas.

Feedback

Aqui vão alguns pontos levantados ao analisar o projeto, é importante lembrar que nem tudo que foi comentado é uma verdade absoluta, e links foram fornecidos para um melhor entendimento a cerca do que foi comentado, espero que esse feedback o ajude a evoluir e se tornar um profissional melhor!

Comentários gerais

Foi utilizado Bootstrap para a construção da interface, ficou bonito porém identifiquei alguns problemas de UX, como poder clicar em um botão que não faz nada e não dá nenhum feedback para o usuário (adicionar um item sem nome por exemplo), os botões de modo geral só deveriam estar habilitados se a ação que eles se propõem a fazer é possível. Também senti falta de um indicador que as operações (comunicaão back <-> front) estão sendo executadas.

Outra coisa que notei, é que listas vazias não são persistidas, criei uma lista, não adicionei nenhum item, fui para a home e ao voltar para a listagem das listas só estavam lá as que continham itens adicionados à elas.

Achei interessante o modo como foi utilizado o :after para fazer a tela de listagem semitransparente na home, inclusive não esperava ser uma imagem, ficou legal.

O projeto não está compatível com AOT, os ngIf por exemplo estão acessando uma dependência privada do componente, os templates não devem acessar propriedades privadas, mesmo que do próprio componente.

Alguns comentários de partes mais específicas do projeto podem ser encontrados pesquisando por FEEDBACK.

Preferivelmente classes CSS são utilizadas no formato kebab-case.

Recursos de estilo utilizados:
